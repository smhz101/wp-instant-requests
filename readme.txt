Floating request/inquiry modal for WooCommerce products (and single posts/pages). Saves entries to a CPT. Admin can reply. Extensible for Dokan/EDD. WPML/Polylang-ready.

== Features ==
* Floating button with modal, blur backdrop
* Auto-fill name/email if logged-in
* Topics dropdown (customizable)
* CPT to store requests; admin reply via metabox (emails user & logs)
* Settings page (button text, color, side, topics, locations, email notify)
* Integration layer (WooCommerce provided; add more easily)
* WPML/Polylang support (text domain + wpml-config.xml + string registration)
